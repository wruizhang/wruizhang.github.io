ABOUT

“The Dream Catcher” is an interactive visualization of dream contents. Inspired by a set of scientific coding rules known as the Hall/Van de Castle System, I attempted to translate a few of its categories — setting, objects, characters and emotions — into visual representations. Using the dream catcher as a visual framework, each of the four categories corresponds to a different component of the dream catcher. The visual representations are a combination of code generated sketches (p5.js) and hand-drawn illustrations.
